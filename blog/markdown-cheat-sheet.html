Markdown Cheat Sheet for Beginners

February 5, 2025 5 min read

This comprehensive markdown cheat sheet covers all the essential syntax you need to format text, create lists, add links, insert images, and more. Perfect for beginners learning markdown or as a quick reference for experienced users. All examples are ready to copy and paste!

Headings

Create headings using hash symbols (#). The number of hashes determines the heading level.

# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

Text Formatting

Format text with bold, italic, strikethrough, and other styles.

**Bold text**
*Italic text*
***Bold and italic***
~~Strikethrough~~
`Inline code`

Result:

Bold text
Italic text
Bold and italic
Strikethrough
Inline code

Lists

Unordered Lists

- Item 1
- Item 2
  - Nested item 2.1
  - Nested item 2.2
- Item 3

Result:

  • Item 1
  • Item 2
    • Nested item 2.1
    • Nested item 2.2
  • Item 3

Ordered Lists

1. First item
2. Second item
3. Third item
   1. Nested item 3.1
   2. Nested item 3.2

Result:

  1. First item
  2. Second item
  3. Third item
    1. Nested item 3.1
    2. Nested item 3.2

Links

[Link text](https://example.com)
[Link with title](https://example.com "Link title")
<https://example.com>

Result:

Link text
Link with title
https://example.com

Images

![Alt text](image-url.jpg)
![Alt text](image-url.jpg "Image title")

Blockquotes

> This is a blockquote
> It can span multiple lines
>
> > Nested blockquote

Result:

This is a blockquote
It can span multiple lines

Nested blockquote

Code Blocks

Inline Code

Use `code` in your sentence.

Code Blocks

```javascript
function hello() {
  console.log('Hello, World!');
}
```

Result:

function hello() {
  console.log('Hello, World!');
}

| Header 1 | Header 2 | Header 3 |

| Cell 4   | Cell 5   | Cell 6   |

Header 1 Header 2 Header 3
Cell 1 Cell 2 Cell 3
Cell 4 Cell 5 Cell 6

| L2   | C2     | R2    |

Escaping Characters

Use backslash (\) to escape markdown characters:

\*Not italic\*
\# Not a heading
\[Not a link\](url)

HTML in Markdown

You can use HTML tags directly in markdown:

<div align="center">
  <strong>Centered bold text</strong>
</div>

<details>
  <summary>Click to expand</summary>
  Hidden content here
</details>

Tips for Learning Markdown

  1. Practice Regularly: The best way to learn is by using it
  2. Use a Preview Tool: Try our Markdown Viewer to see results instantly
  3. Start Simple: Begin with basic formatting and gradually add more
  4. Keep This Cheat Sheet Handy: Bookmark this page for quick reference
  5. Read Other People's Markdown: Check out README files on GitHub

Common Mistakes to Avoid

  • Missing Blank Lines: Always add blank lines before and after headings, lists, and code blocks
  • Inconsistent Spacing: Be consistent with indentation in nested lists
  • Forgetting Alt Text: Always include alt text for images
  • Not Escaping Special Characters: Use backslash to escape markdown characters when needed

Conclusion

This cheat sheet covers all the essential markdown syntax you need for most writing tasks. Markdown is designed to be simple and intuitive, so don't be intimidated by the number of options. Start with the basics (headings, bold, italic, lists, and links) and gradually incorporate more advanced features as you become comfortable.

Remember, the best way to learn markdown is by using it. Try our Markdown Viewer to practice and see your markdown rendered in real-time. Happy writing!
