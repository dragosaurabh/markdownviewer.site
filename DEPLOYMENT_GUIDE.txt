DEPLOYMENT STEPS
================================================================================

STEP 1: PREPARE YOUR HOSTING
-----------------------------
□ Access your shared hosting control panel (cPanel, Plesk, etc.)
□ Navigate to File Manager or use FTP client
□ Locate your public_html or www directory

STEP 2: UPLOAD FILES
--------------------
□ Upload ALL files maintaining the directory structure:
  
  public_html/
  ├── index.html
  ├── editor.html
  ├── blog.html
  ├── .htaccess
  ├── robots.txt
  ├── sitemap.xml
  ├── blog/
  │   ├── 10-reasons-markdown-documentation.html
  │   ├── markdown-vs-rich-text-editors.html
  │   ├── convert-markdown-to-pdf.html
  │   ├── readme-best-practices.html
  │   └── markdown-cheat-sheet.html
  ├── css/
  │   ├── main.css
  │   ├── themes.css
  │   ├── markdown.css
  │   ├── blog.css
  │   ├── dark-mode-fixes.css
  │   └── editor.css
  ├── js/
  │   ├── app.js
  │   ├── ui.js
  │   ├── utils.js
  │   └── [all other JS files]
  └── assets/
      └── examples/
          └── sample.md

IMPORTANT: Make sure .htaccess is uploaded (it's hidden by default)

STEP 3: VERIFY .HTACCESS
-------------------------
□ Ensure .htaccess is in the root directory
□ Check file permissions: 644 (rw-r--r--)
□ If using Nginx instead of Apache, convert rules to nginx.conf

STEP 4: TEST THE DEPLOYMENT
----------------------------
□ Visit your domain: https://yourdomain.com
□ Test homepage loads correctly
□ Test navigation links work
□ Test dark mode toggle
□ Test markdown viewer functionality:
  - Upload a .md file
  - Paste markdown text
  - Load from URL
□ Test blog posts load correctly
□ Test on mobile devices

STEP 5: VERIFY SEO FILES
-------------------------
□ Check robots.txt: https://yourdomain.com/robots.txt
□ Check sitemap.xml: https://yourdomain.com/sitemap.xml
□ Submit sitemap to Google Search Console
□ Submit sitemap to Bing Webmaster Tools

================================================================================
SERVER REQUIREMENTS
================================================================================

MINIMUM REQUIREMENTS:
✓ Web server: Apache 2.4+ or Nginx 1.18+
✓ PHP: NOT required (static site)
✓ Database: NOT required
✓ HTTPS: Recommended (for security)
✓ Gzip compression: Recommended (for performance)

RECOMMENDED APACHE MODULES:
✓ mod_rewrite (for .htaccess URL rewriting)
✓ mod_deflate (for gzip compression)
✓ mod_expires (for browser caching)
✓ mod_headers (for security headers)

================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: 404 errors on pages
SOLUTION: 
- Verify .htaccess is uploaded
- Check mod_rewrite is enabled
- Verify file permissions (644 for files, 755 for directories)

ISSUE: CSS/JS not loading
SOLUTION:
- Check file paths are correct
- Verify css/ and js/ directories uploaded
- Check browser console for errors
- Clear browser cache

ISSUE: Dark mode not working
SOLUTION:
- Check js/dark-mode.js is uploaded
- Verify browser console for JavaScript errors
- Test in different browsers

ISSUE: Blog posts not loading
SOLUTION:
- Verify blog/ directory uploaded
- Check all 5 blog HTML files are present
- Test direct URLs: /blog/10-reasons-markdown-documentation.html

ISSUE: Markdown viewer not working
SOLUTION:
- Check all JS files in js/ directory uploaded
- Verify browser console for errors
- Test with sample markdown text
