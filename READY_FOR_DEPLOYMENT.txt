‚ö†Ô∏è IMPORTANT NOTES
================================================================================

1. .HTACCESS FILE
   - This file is CRITICAL for the site to work
   - It's hidden by default in file managers
   - Make sure it's uploaded to the root directory
   - File permissions should be 644

2. DIRECTORY STRUCTURE
   - Maintain the exact directory structure
   - Don't rename directories (css, js, blog, assets)
   - All paths are relative and will break if changed

3. FILE PERMISSIONS
   - Files: 644 (rw-r--r--)
   - Directories: 755 (rwxr-xr-x)
   - .htaccess: 644

4. HTTPS
   - Strongly recommended for security
   - Most hosting providers offer free SSL (Let's Encrypt)
   - Update URLs in sitemap.xml to https:// after enabling
