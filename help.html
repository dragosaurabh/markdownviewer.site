Help & Frequently Asked Questions

Getting Started

Welcome to Markdown Viewer! This guide will help you get started with viewing and converting your markdown files online. Whether you're new to markdown or an experienced user, you'll find everything you need to know right here.

Quick Start Guide

  1. Choose Your Input Method: Select from URL, Upload, or Text tabs at the top of the page
  2. Load Your Markdown: Paste a URL, upload a file, or type/paste markdown text
  3. View Instantly: Your markdown is rendered immediately with beautiful formatting
  4. Customize: Adjust font size, content width, or enable dark mode
  5. Export: Download as HTML, copy to clipboard, or share with others

Frequently Asked Questions

How do I view a markdown file online?

Viewing markdown files with Markdown Viewer is simple and can be done in three ways:

  • URL Method: Click the "URL" tab, paste the direct link to your markdown file (like a GitHub raw URL), and click "Load". The file will be fetched and rendered instantly.
  • Upload Method: Click the "Upload" tab, then either drag and drop your .md file onto the upload area or click "Choose File" to browse your computer. Files up to 5MB are supported.
  • Text Method: Click the "Text" tab and paste or type your markdown content directly into the text area. The preview updates automatically as you type.

Is my markdown file data secure?

Absolutely! Security and privacy are our top priorities. All markdown processing happens entirely in your web browser using client-side JavaScript. This means:

  • Your files never leave your device
  • We don't upload, store, or have any access to your content
  • No server communication is required for file upload or text input
  • Even when fetching from URLs, the content is retrieved directly by your browser

You can even use Markdown Viewer offline (after the initial page load) for the upload and text input features.

What markdown syntax is supported?

Markdown Viewer supports full GitHub Flavored Markdown (GFM), which includes:

  • Basic Formatting: Headers (H1-H6), bold, italic, strikethrough
  • Links and Images: Inline links, reference links, images with alt text
  • Lists: Ordered lists, unordered lists, nested lists
  • Code: Inline code, fenced code blocks with syntax highlighting for 200+ languages
  • Tables: Full table support with alignment options
  • Blockquotes: Single and nested blockquotes
  • Task Lists: Checkboxes for to-do lists
  • Horizontal Rules: Dividers using ---, ***, or ___
  • HTML: Safe HTML rendering (sanitized for security)

For a complete syntax guide with examples, visit our Markdown Guide.

How do I export my rendered markdown?

Markdown Viewer offers several export options:

  • Download as HTML: Click the download icon in the toolbar to save a standalone HTML file that includes all styling and can be opened in any browser.
  • Copy HTML: Click the copy icon to copy the rendered HTML to your clipboard for pasting into other applications.
  • Print/PDF: Use your browser's print function (Ctrl+P or Cmd+P) and select "Save as PDF" to create a PDF version.
  • Share Link: Click the share icon to generate a shareable URL that includes your markdown content (note: links may be long for large documents).

Can I use Markdown Viewer offline?

Yes, partially! Once you've loaded the Markdown Viewer page, you can use the file upload and text input features completely offline. The URL fetching feature requires an internet connection since it needs to retrieve files from external sources. The page uses browser caching, so subsequent visits load faster.

What file formats are supported?

Markdown Viewer accepts files with the following extensions:

  • .md (standard markdown files)
  • .markdown (alternative markdown extension)
  • .txt (plain text files with markdown syntax)

The maximum file size is 5MB. If you need to view larger files, consider breaking them into smaller sections or using the text input method to paste specific portions.

How do I enable dark mode?

Dark mode can be enabled in two ways:

  • Automatic: Markdown Viewer automatically detects your system's dark mode preference and applies it.
  • Manual Toggle: Click the sun/moon icon in the top-right corner of the page to manually switch between light and dark modes. Your preference is saved locally.

Dark mode applies to both the interface and the rendered markdown content for comfortable reading in low-light conditions.

What is the Table of Contents feature?

The Table of Contents (TOC) is automatically generated from the headers in your markdown document. It appears in the left sidebar (on desktop) and provides:

  • Quick navigation to any section of your document
  • Hierarchical structure showing document organization
  • Active highlighting of the current section as you scroll
  • Collapsible on mobile devices to save screen space

The TOC is generated automatically whenever you load markdown content with headers (H1, H2, H3, etc.).

How do I search within my markdown document?

Markdown Viewer includes a powerful search feature:

  • Click the search icon in the toolbar or press Ctrl+F (Cmd+F on Mac)
  • Type your search query in the search box
  • Use the arrow buttons or press Enter to navigate between matches
  • Press Shift+Enter to go to the previous match
  • Press Escape to close the search bar

Search results are highlighted in the document, and the match counter shows your current position.

What is Reading Mode?

Reading Mode is a distraction-free viewing experience that:

  • Hides all UI elements except the content
  • Optimizes typography for comfortable reading
  • Provides font size controls
  • Shows reading time estimate
  • Can be activated by clicking the reading mode icon in the toolbar
  • Exit by pressing Escape or clicking the X button

Reading Mode is perfect for reviewing long documents or documentation.

Can I customize the appearance?

Yes! Markdown Viewer offers several customization options:

  • Font Size: Choose from Small, Medium, or Large in the toolbar
  • Content Width: Select Narrow, Medium, or Wide to adjust reading width
  • Dark Mode: Toggle between light and dark themes
  • Syntax Theme: Code blocks use a professional syntax highlighting theme

All preferences are saved locally in your browser and persist across sessions.

How do I load markdown from GitHub?

To view markdown files from GitHub:

  1. Navigate to the markdown file on GitHub
  2. Click the "Raw" button to view the raw markdown
  3. Copy the URL from your browser's address bar
  4. Paste the URL into Markdown Viewer's URL tab
  5. Click "Load" to render the file

Example URL format: https://raw.githubusercontent.com/username/repo/branch/file.md

Why isn't my markdown rendering correctly?

If your markdown isn't rendering as expected, check these common issues:

  • Syntax Errors: Ensure your markdown syntax is correct (see our Markdown Guide)
  • File Encoding: Files should be UTF-8 encoded
  • URL Access: If loading from URL, ensure the file is publicly accessible and not blocked by CORS
  • File Size: Files larger than 5MB cannot be uploaded
  • Browser Compatibility: Use a modern browser (Chrome, Firefox, Safari, or Edge)

If problems persist, try the text input method to isolate the issue.

What happens to my markdown content?

Your markdown content is processed entirely in your browser and never sent to our servers. Here's what happens:

  • File Upload: The file is read by your browser and processed locally
  • Text Input: Text stays in your browser's memory
  • URL Fetch: Your browser fetches the file directly from the source
  • Local Storage: Only your preferences (dark mode, font size) are saved locally

We have no access to your content and don't track what you view. See our Privacy Policy for details.

Can I convert markdown to PDF?

Yes! While we don't have a direct "Export to PDF" button, you can easily create PDFs:

  1. Load your markdown file in Markdown Viewer
  2. Press Ctrl+P (Windows/Linux) or Cmd+P (Mac) to open the print dialog
  3. Select "Save as PDF" or "Microsoft Print to PDF" as the destination
  4. Adjust settings if needed (margins, headers/footers)
  5. Click "Save" or "Print" to create your PDF

The PDF will include all formatting, images, and styling from the rendered markdown. For more details, check our blog post on converting markdown to PDF.

Why can't I load a markdown file from a URL?

If you're having trouble loading markdown from a URL, common causes include:

  • CORS Restrictions: The server hosting the file may not allow cross-origin requests. This is a security feature of web browsers.
  • Private/Protected Files: The file may require authentication or be in a private repository.
  • Incorrect URL: Ensure you're using the direct link to the raw markdown file, not the HTML page displaying it.
  • Network Issues: Check your internet connection and try again.

Solution: If URL loading fails, download the file and use the Upload tab instead.

Troubleshooting

Common Issues and Solutions

Issue: Page won't load or is very slow

Solutions:

  • Clear your browser cache and reload the page
  • Check your internet connection
  • Try a different browser
  • Disable browser extensions that might interfere

Issue: Syntax highlighting not working

Solutions:

  • Ensure you've specified the language in your code block (e.g., ```javascript)
  • Check that the language name is spelled correctly
  • Wait a moment for the syntax highlighter to load
  • Try refreshing the page

Issue: Images not displaying

Solutions:

  • Verify the image URL is correct and publicly accessible
  • Check if the image host allows hotlinking
  • Ensure the image URL uses HTTPS (not HTTP)
  • Try opening the image URL directly in your browser

Issue: Dark mode not working

Solutions:

  • Click the sun/moon icon to manually toggle dark mode
  • Check your browser's local storage isn't disabled
  • Clear your browser cache and try again
  • Ensure JavaScript is enabled in your browser
